# Indian Stock Market Analysis
This project focuses on analyzing Open Interest (OI) data of NIFTY options and identifying the Max Pain point to understand market sentiment.  
It also includes a script to scrape the list of NIFTY 50 stocks dynamically (without hardcoding).

Task 1 — Fetch NIFTY 50 Stock List
Create a scraping script to fetch the list of NIFTY 50 stocks from a reliable public source (like Wikipedia).  
The script should not hardcode the stock names and should output the list of symbols to a text file (`nifty50_symbols.txt`).

Task 2 — Open Interest (OI) Analysis and Max Pain Identification
Using the NIFTY 50 list obtained from Task 1
